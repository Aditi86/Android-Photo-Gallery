Home Page: 

User can create album by pressing the create button. 
After album is created user can select the album to open, rename, delete the album.
Search button will help user to search for tags 
	The searched tag exits the the photos will be shown in grid view. 
	Click the gridView photo to see it's tags




Open Album:

When user opnes album, user can add the image by add button.
User can click the image in album to Open, delete, Move. 
	To Move Image:
		Click the image and select Move
		User can then go to another album and can press Move Here button to move the image. 
	Open Image:
		Whne user opens the image, user can then add tags, and see images in slideView.
		once the tag is added user can select that tag to delete by pressing Delete Tag button.
